# 快速开始

## 1. 准备 Termux

- 推荐安装 [GitHub 版](https://github.com/termux/termux-app/releases) 或 [F-Droid 版](https://f-droid.org/packages/com.termux/)
- **请勿使用 Google Play 版，可能导致脚本异常运行！**

| 渠道    | 版本                    | 下载地址                                                                                  |
|---------|-------------------------|:----------------------------------------------------------------------------------------:|
| GitHub  | v0.118.3                | [点击下载](https://github.com/termux/termux-app/releases/download/v0.118.3/termux-app_v0.118.3+github-debug_universal.apk) |
| GitHub  | v0.119.0-beta.3（测试） | [点击下载](https://github.com/termux/termux-app/releases/download/v0.119.0-beta.3/termux-app_v0.119.0-beta.3+apt-android-7-github-debug_universal.apk) |
| F-Droid | v0.118.3                | [点击下载](https://f-droid.org/repo/com.termux_1002.apk)                                  |
| F-Droid | v0.119.0-beta.3（测试） | [点击下载](https://f-droid.org/repo/com.termux_1022.apk)                                  |

> 如遇下载问题，可加入 QQ 群 `807134015` 获取群文件。

---

## 2. 一键安装 SillyTavern-Termux

打开 Termux，复制并粘贴以下命令后回车：

```bash
curl -O https://raw.githubusercontent.com/print-yuhuan/SillyTavern-Termux/refs/heads/main/Install.sh && bash Install.sh
```

- 首次运行时会自动申请存储权限，请务必选择“允许”。
- 全流程自动执行，无需手动输入其他命令。

---

## 3. 启动与管理

- 安装完成后，**重启 Termux** 即可自动进入主菜单。
- 所有操作均可通过数字菜单完成，无需记忆命令。

---

## 4. 更新与卸载

- 通过菜单 `6. 脚本管理` 可实现脚本自更新、日志查看及一键卸载。
